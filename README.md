# HISTORYSOLDS

## Расширение браузера для импорта истории

В папке `extension/` лежит минимальное расширение (Manifest V3), которое добавляет кнопки импорта сделок:

- **CS.MONEY**: https://cs.money/ru/market/history/
- **Market.CSGO**: https://market.csgo.com/ru/usercab/history/market

Расширение складывает сделки в очередь импорта, а когда вы открываете
`cs_profit_calc_history_editable_v5.html`, они автоматически добавляются
в историю калькулятора.

### Установка (Chrome/Chromium)

1. Откройте `chrome://extensions`.
2. Включите **Режим разработчика**.
3. Нажмите **Загрузить распакованное** и выберите папку `extension/`.
4. Для работы с локальным файлом включите опцию **Разрешить доступ к файлам URL** у расширения.
